1) Eclipse Project (TicTacToeMinMaxMoveUtilTreeGenerator_00001.rar) with
a Java 6 application that generates a Min/Max move utility tree for
the game of Tic-Tac-Toe. The move utility tree is saved both as a
text file and as a Java serialized object.

2) Targets: Eclipse Helios 3.6, Java 6

3) To start: 
   a) unzip into a folder 
   b) open in Eclipse 
   c) Run as Java application
